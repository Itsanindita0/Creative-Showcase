Creative Showcase — MERN Image Sharing Platform
A full-stack image sharing web application where users can sign up, log in, upload images, delete
their uploads, and view public galleries.
Built using the MERN stack and deployed — suitable as a project assignment or portfolio project.
Features
- User Authentication: Signup, Login using JWT
- Upload Images to Cloudinary
- User Dashboard to manage uploads
- Public Gallery showing random images
- User Profiles showing each user's images
- Secure Access with JWT
- Fully Responsive UI with Tailwind CSS
Tech Stack
Frontend: React + Vite, Tailwind CSS, React Router DOM, Axios
Backend: Node.js, Express.js, MongoDB, JWT, Multer, Cloudinary
